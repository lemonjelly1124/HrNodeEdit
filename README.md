# 简介

Vscode 的 节点编辑器插件，提供React 实现


# 运行

```bash
git clone https://github.com/lemonjelly1124/HrNodeEdit.git
```

使用 vscode 打开项目，运行：

（如果没有安装 pnpm）

```bash
npm i pnpm -g
```

```	bash
pnpm install
pnpm dev
```

脚本执行完成后：

- 按 F5 开启调试
- 调试窗口打开后，点击 activitybar 上的图标。

# 特点

- 完整示例：提供 Vue 和 React 的完整示例
- 解决方案：资源路径、通讯、架构等问题的解决方案一应俱全
- 拿来即用：可运行、架构完善、易扩展、生产级的示例项目
- 文档详细：记录每一个踩过的坑
